# Project Notes - Media Landing Page

## Development Summary

We tried to reach the design as much as we can and not pixel perfect as we not have something like figma.

**Key Features & Modern Angular:**
- Used modern Angular features for showing the loops (@for, @if, standalone components, signals where applicable).
- Responsive clamp() units, CSS grid/flex modern layout.
- Smooth CSS transitions/keyframes for animations (solutions cards hover/interval wider scale).

**Interactive Elements:**
- Added the interactive stats component to use Three.js (3D graphs with floating statues/particles).
- Solutions cards: Wider scale on hover and staggered interval pulse.

**Assets:**
- Used CDN images for the images needed (Unsplash for featured solution image).

**Tech Stack:**
- Angular 17 standalone.
- Three.js for 3D interactions.
- Pure CSS animations/transitions where possible.

**Run:**
`ng serve`
